## 线路到底有什么区别：Premium vs Lite别搞混

DMIT的套餐分两条线，这是选购前必须搞清楚的事。

**Premium（高端线路）**

回程走CN2 GIA或者AS929，这两条都是对大陆优化的精品线路。CN2 GIA是电信的顶级商用线路，延迟低、丢包少；AS9929是联通的优质骨干网。Premium套餐的去程也做了优化，不是单纯的普通路由。

实际体验：洛杉矶到上海，晚高峰延迟稳在150-170ms区间，丢包基本可以忽略。跑个speedtest，国内方向的速度相当能打。

**Lite（标准线路）**

走普通线路，价格便宜一截。如果你的业务不依赖国内访问速度，或者主要面向海外用户，Lite够用。但如果你在意国内体验，Lite和Premium的差距在晚高峰会很明显。

我自己用的是Premium。Lite我测过朋友的机器，白天差距不大，晚上8点以后就能感受到了。

## 踩坑提醒：这几点买之前要想清楚

**流量超了怎么办**

DMIT的流量超额之后不是直接断网，而是限速。具体限速策略看套餐说明，但如果你的业务流量波动大，建议选高一档的套餐，别卡在边缘。

**Lite和Premium别买错**

这是最常见的踩坑。有人图便宜买了Lite，用了一周发现晚高峰体验差，想换Premium还要重新开机器、迁数据。买之前想清楚自己的主要用途是面向国内还是海外。

**洛杉矶机房的物理延迟**

美国西海岸到中国大陆，物理距离决定了延迟下限大概在130ms左右，这是绕不过去的。如果你需要更低延迟，DMIT的香港节点是另一个选项，但价格更高。

**控制面板和技术支持**

DMIT用的是自家控制面板，功能够用，重装系统、重启、查流量都没问题。技术支持走工单，响应速度中规中矩，不是那种秒回的，但问题基本都能解决。
